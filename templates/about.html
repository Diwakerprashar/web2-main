Weather forecasting is the prediction of the state of the atmosphere for a given location using the application of science and technology. This is the final Project of the Industrial training that we undertook at NIELIT Ropar/Chandigarh in colloboration with IIT Ropar. Technologies used in this field are Frontend: (HTML CSS BOOTSTRAP) & Backend: (Python(Data Science , Machine Learning) Flask SQlite3). The Weather Forecast model is trained using Machine Learning.
